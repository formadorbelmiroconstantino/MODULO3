Alinhando o Conteúdo em Tabelas

Para alinhar o conteúdo em tabelas HTML, a prática recomendada é utilizar propriedades CSS, pois atributos antigos de HTML (como align ou valign) são considerados obsoletos.

1. Alinhamento Horizontal

Utilize a propriedade text-align nas tags <th> ou <td>:

  • left: Alinha o conteúdo à esquerda (padrão para <td>).
  • center: Centraliza o conteúdo (padrão para <th>).
  • right: Alinha o conteúdo à direita.

2. Alinhamento Vertical

Utilize a propriedade vertical-align para controlar a posição do conteúdo em relação à altura da célula:

  • top: Alinha o conteúdo ao topo da célula.
  • middle: Centraliza verticalmente (comportamento padrão na maioria dos navegadores).
  • bottom: Alinha o conteúdo à base da célula.

3. Centralização da Tabela Inteira

Para centralizar a tabela como um todo na página, aplique margin: auto ao elemento <table> ou utilize Flexbox no elemento pai com justify-content: center.

Exemplo (HTML):

<table>
<th>A</th>
<th>B</th>
<th>C</th>
<tr>
<td>A1</td>
<td>B1</td>
<td>C1</td>
</tr>
<tr>
<td>A2</td>
<td>B2</td>
<td>C2</td>
</tr>
<tr>
<td>A3</td>
<td>B3</td>
<td>C3</td>
</tr>
<tr>
<td>A4</td>
<td>B4</td>
<td>C4</td>
</tr>
</table>

Exemplo (CSS):

table {
width: 100%; border-collapse: collapse; /*Remove o espaco entre as celulas*/
} th, td {
border: 1px solid #ccc;
padding: 10px;
/* Alinhamentos */
text-align: center; /* Horizontal */
vertical-align: middle; /* Vertical */
}

Resultado:

A B C
A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

Observação: Para layouts mais complexos ou responsivos, você pode explorar o uso de CSS Flexbox ou Grid para organizar o conteúdo interno das células.
