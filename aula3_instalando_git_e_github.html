Instalando Git e GitGub no PC

Para instalar o Git e configurar o GitHub no seu PC, siga os passos abaixo para o seu sistema operacional:

1. Instalando o Git

O Git é a ferramenta de linha de comando que gerencia as versões dos seus arquivos.

  • Windows: Baixe o instalador no site oficial do Git, clique em Install e Click here to download. Execute o arquivo .exe, mantenha as opções padrão ("Next") e certifique-se de que a opção "Git Bash" esteja selecionada.
  • macOS: O método mais simples é abrir o Terminal e digitar git. Se não estiver instalado, o sistema solicitará a instalação das ferramentas de linha de comando do Xcode. Alternativamente, use o instalador binário.
  • Linux (Debian/Ubuntu): Use o comando sudo apt install git no terminal.

Observação importante:Durante a instalação, fique atento na hora de seleccionar o editor de códigos. Lembre-se que estamos a utilizar o VSCode, no entanto, não seleccione o Insiders, seleccione a outra versão por ser a estável.

2. Configurando o GitHub Desktop (Interface Gráfica)

Para quem prefere não usar apenas comandos, o GitHub Desktop facilita a sincronização.

  1. Baixe em desktop.github.com.
  2. Após instalar, faça login com sua conta do GitHub para vincular o PC ao seu perfil.

3. Configuração Inicial via Terminal

  1. Após instalar o Git, você deve identificar-se para que seus registros de alterações (commits) tenham seu nome:
  2. Abra o Git Bash (Windows) ou Terminal (Mac/Linux).
  3. Digite os comandos abaixo substituindo pelos seus dados:
    • git config --global user.name "Seu Nome"
    • git config --global user.email "seuemail@exemplo.com"
