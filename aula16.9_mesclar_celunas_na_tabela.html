Mesclagem de Células de Tabelas

Em HTML, “mesclar células” numa tabela é feito com atributos do próprio HTML, não com CSS:

  • colspan → mescla colunas (horizontal)
  • rowspan → mescla linhas (vertical)

O CSS entra só para estilo (bordas, alinhamento, cores, etc.).

1. Mesclar colunas (colspan)

Relatório de Vendas
Mês Produto Total
Jan Diamante A 120
  • O cabeçalho “Relatório de Vendas” ocupa 3 colunas.

2. Mesclar linhas (rowspan)

Departamento Funcionário Estado
TI Ana Activo
Paulo Activo
RH Maria Inactivo
  • “TI” ocupa 2 linhas.

CSS básico (só estilo)

.t{
width:100%;
border-collapse:collapse;
}
.t th, .t td{
border:1px solid #ddd;
padding:10px;
text-align:left;
}
.t th{
background:#f5f5f5;
}

Regras importantes (para não “quebrar” a tabela)

  • Se usas colspan="3", aquela célula substitui 3 células daquela linha.
    • Se usas rowspan="2", na linha seguinte tu não deves repetir a célula “ocupada”. Planeia a “grade” da tabela como se fosse um grid: cada mesclagem “consome espaço”.
