Agrupando Colunas com colgroup

A tag HTML <colgroup> agrupa colunas em uma tabela para aplicar estilos CSS (como background-color ou width) de forma eficiente, sem repetir regras em cada célula <td>. Ela é colocada logo após <table> e pode conter tags <col> individuais ou usar o atributo span para abranger múltiplas colunas.

Principais Pontos:

  • <colgroup>: Define o grupo.
  • <col>: Define as propriedades de uma ou mais colunas dentro do grupo.
  • Atributo span: Define quantas colunas consecutivas o <col> abrange (ex: <col span="2">).
  • Limitação: Apenas algumas propriedades CSS funcionam com <col>, principalmente width, background, border e visibility.
