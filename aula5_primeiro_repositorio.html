Criando o Primeiro Repositório

Para criar um repositório usando o GitHub Desktop, siga estes passos:

Criar um Novo Repositório Local

  1. Abra o aplicativo GitHub Desktop.
  2. No menu superior, clique em File (Arquivo) e selecione New repository... (ou use o atalho Ctrl + N no Windows / Cmd + N no Mac).
  3. Preencha os campos na janela que aparecer:
      1. Name: O nome do seu projeto (recomenda-se letras minúsculas e hífens em vez de espaços).
      2. Description: Uma breve explicação (opcional).
      3. Local path: A pasta no seu computador onde o projecto será salvo.
      4. Initialize this repository with a README: Marque esta opção para criar um arquivo de apresentação inicial.
      5. Git ignore: Escolha um modelo se quiser que o Git ignore arquivos específicos (ex: arquivos de sistema ou bibliotecas pesadas).
      6. License: Selecione uma licença (como MIT) se o projecto for aberto.
  4. Clique em Create repository.

2. Publicar no GitHub (Nuvem)

Após criar o repositório no seu computador, ele ainda não está na internet. Para enviá-lo, siga os passos seguintes:

  1. No painel principal do GitHub Desktop, clique no botão azul Publish repository. Além do botão azul, aparece uma mensagem com o título Publish your repository to GitHub.
  2. Verifique o nome e se deseja manter o código privado (marcando a caixa Keep this code private). Se deixar marcado, significa que as outras pessoas que consultarem o seu perfil do GitHub não terão acesso ao projecto que estiver a publicar. Se quiser que outras pessoas tenham acesso, desmarque essa opção.
  3. Clique em Publish Repository.
