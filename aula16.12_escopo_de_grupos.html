Escopo de Grupos em Tabelas

Para gerenciar escopos de grupo em tabelas grandes com HTML e CSS, utilize as tags estruturais <thead>, <tbody> e <tfoot> para agrupar linhas, e o atributo scope (col, row, colgroup, rowgroup) nas tags <th> para definir a relação das células. O CSS permite estilizar esses grupos para melhorar a legibilidade de dados complexos.

Principais Técnicas e Elementos:

  • Estruturação com <thead>, <tbody> e <tfoot>: Separa o cabeçalho, corpo e rodapé, facilitando a aplicação de estilos CSS a seções específicas de grandes conjuntos de dados.
  • Atributo scope em <th>:
    • scope="col": Define que o cabeçalho se refere a toda a coluna.
    • scope="row": Define que o cabeçalho se refere a toda a linha.
    • scope="colgroup": Define um cabeçalho para um grupo de colunas.
    • scope="rowgroup": Define um cabeçalho para um grupo de linhas.
  • Mesclagem de Células: Utilize colspan para colunas e rowspan para linhas para agrupar células adjacentes em tabelas complexas.
  • CSS para Tabelas Grandes:
    • Use seletores nth-child() para criar linhas listradas (zebra striping) para facilitar a leitura.
    • Aplique border-bottom nos elementos e para destacar divisões.
    • Utilize a propriedade `position: sticky` no CSS em para manter o cabeçalho fixo durante a rolagem de tabelas grandes.

Exemplo Prático de Estrutura:

No HTML

<table border="1" class="t">
<caption>Relatório Mensal</caption>
<thead>
<tr>
<th scope="col">Produto</th>
<th scope="col">Qtd</th>
</tr>
</thead>
<tbody>
<tr>
<th scope="rowgroup" rowspan="3">Eletrónicos</th>
<td>TV</td>
</tr>
<tr>
<td>Telemómeis</td>
</tr>
<tr>
<td>Computadores Portáteis</td>
</tr>
</tbody>
<tfoot>
<tr>
<td colspan="2">Total Geral</td>
</tr>
</tfoot>
</table>

No CSS

.t {
width: 100%;
border-collapse: collapse;
}

td {
padding: 10px;
text-align: left;
text-align: center;
}

.t th {
background: #f5f5f5;
}

Resultado

Produto Qtd
