Imagens que se Repetem no Fundo do Site

Imagens que se repetem no fundo de um site são controladas pelo CSS com a propriedade background-repeat, que define o padrão de repetição, sendo repeat (padrão, repete em X e Y), repeat-x (só horizontal), repeat-y (só vertical) e no-repeat (não repete) os valores mais comuns, permitindo que o desenvolvedor crie texturas ou fundos sem ladrilhos parciais.

Valores Comuns de background-repeat:

  • repeat: A imagem se repete tanto horizontalmente quanto verticalmente, preenchendo todo o espaço (comportamento padrão).
  • repeat-x: A imagem se repete apenas na direção horizontal (eixo X).
  • repeat-y: A imagem se repete apenas na direção vertical (eixo Y).
  • no-repeat: A imagem é exibida apenas uma vez, sem nenhuma repetição, preenchendo o espaço ou não.
  • space: A imagem é repetida, e o espaço extra é distribuído entre as repetições, sem cortar as imagens.
  • round: A imagem é repetida para preencher o espaço, redimensionando-se para que não fiquem ladrilhos parciais.

Exemplo

body {
background-image: url("sua-imagem.png");
background-repeat: no-repeat; /* Para não repetir */
/* Ou */
/* background-repeat: repeat-x; */ /* Para repetir só na horizontal */
/* Ou */
/* background-repeat: repeat-y; */ /* Para repetir só na vertical */
}

Controlar a Imagem de Fundo:

  • background-size: Define o tamanho da imagem (ex: cover, contain, largura altura).
  • background-position: Define a posição inicial da imagem (ex: center, top left).
  • background-attachment: Fixa a imagem no fundo enquanto o conteúdo rola (pode receber fixed ou scroll).
