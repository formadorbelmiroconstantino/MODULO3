Caption e Escopo de Títulos em Tabelas

Para criar um efeito zebrado (linhas alternadas) em tabelas HTML, utiliza-se CSS com o selector tr:nth-child(even) (para linhas pares) ou tr:nth-child(odd) (para linhas ímpares) dentro da tag <tbody>. Essa técnica melhora a legibilidade dos dados ao alternar cores de fundo, como cinza claro e branco. Uma das vantagens do DataTables é que ela já vem com a implementação do Efeito Zebrado.

Exemplo no CSS:

table {
width: 100%;
border-collapse: collapse;
}
table tr:nth-child(even) {
background-color: #f2f2f2; /* Cor para linhas pares */
}
table tr:nth-child(odd) {
background-color: #ffffff; /* Cor para linhas ímpares */
}

Se quisermos simplificar, ao invés de utilizarmos nth-child(even) e nth-child(odd), podemos utilizar nth-child(2n) se quisermos que que as cores variem de uma em uma linha, ou seja, linha sim, linha não. No entanto, só utilizaríamos a cor branca e a cor que definirmos.

Se o fundo do site/projecto for branco, então não precisa utilizar o even e odd. No entanto, se no fundo houver uma imagem, aí sim, pode utilizar o even e odd adicionando cores diferentes de branco entre eles desde que as cores façam parte da paleta de cores do seu projecto. Está ler muito a palavra cores não? Sim, é mesmo para perceber. Essa redundância é muito importante.

  • tr:nth-child(even): Selecciona todas as linhas pares da tabela e aplica uma cor de fundo.
  • tr:nth-child(odd): Selecciona as linhas ímpares para uma cor diferente, se necessário.
  • border-collapse: collapse: Garante que as bordas da tabela sejam unificadas para um design melhor.

O uso de nth-child com CSS3 é preferível por ser mais eficiente e directo, evitando scripts adicionais para criar o efeito.
